This repository contains the computational engine, spreadness analysis suite, and the raw verification data establishing precise extremal lower bounds for the Erd≈ës Sunflower Conjecture when $k=3$. 

By empirically observing maximal sunflower-free families for varying dimensions ($n \in \{3,4,5,6\}$), this research provides verifiable proof of the **Decay of the Normalized Spreadness Constant** ($C = \frac{\kappa}{\log_2 n}$). 

The data robustly demonstrates that the logarithmic scaling requirement ($\log n$) established in recent upper-bound proofs (ALWZ, Rao, Stoeckl) is a methodological artifact of probabilistic union bounds, and not an intrinsic combinatorial property of $n$-uniform subset families. These results present strong computational evidence supporting Erd≈ës's $C^n$ formulation and isolate the mathematical limit required to close the $1,000 proof.

## Repository Structure

### 1. The Core Engines
* **`sunflower_sa_refined.py`**: The primary bitmask-accelerated Simulated Annealing search engine. Uses a specialized min-conflicts delta-update to map the extremums of high-dimension subsets. 
* **`sunflower_engine.py`**: A deterministic exact solver using SAT multi-round batch cutting frameworks for absolute verification on miniature configurations.

### 2. Analysis & Verification
* **`sunflower_analysis.py`**: 100% independent deterministic tool for auditing the output arrays. Performs an $O(m^3)$ check for exact 3-sunflower configurations and calculates the worst-case exact Theoretical Spreadness ($\kappa$) using the formal ALWZ formulas.
* **`analyze_latest.py`**: Pipeline wrapper to feed output datasets into the analysis engine.

### 3. Proof Data (`/proofs/`)
The exact topological configurations generated by the search engine. These JSON arrays serve as verified lower bounds for $f(n, 3)$:
* **`proofs/proof_n2_m6.json`**: Empirically verified exact limit family of 6 sets ($n=2$, computed $\kappa \approx 2.44$).
* **`proofs/proof_n4_m36.json`**: Empirically verified sunflower-free family of 36 sets ($n=4$, computed $\kappa \approx 2.45$).
* **`proofs/proof_n5_m60.json`**: Empirically verified sunflower-free family of 60 sets ($n=5$, computed $\kappa \approx 2.22$).

*(Note: Deeper explorations into $n=5$ limits and bounded $n=6, m=120$ zero-energy states are ongoing. Current computed scaling tables are available in the manuscript).*

## Running the Engine

### Initiating a Search
You can initialize the SA heuristic engine to map local limits yourself using:
```bash
python3 sunflower_sa_refined.py -n <dimension> -m <target_size> -U <universe_size> --steps <anneal_cycles>
```
*Example (verifying the established n=4 boundary):*
```bash
python3 sunflower_sa_refined.py -n 4 -m 36 -U 30 --steps 1000000 
```

### Analytical Verification of Results
To objectively scan any JSON array or raw array printout for 3-sunflowers and to calculate its inverse spreadness metric ($\kappa$) relative to the ALWZ benchmark:
```bash
cat proofs/proof_n5_m60.json | python3 sunflower_analysis.py
```
